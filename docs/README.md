## 처리할 에러 목록

-----

- 길이가 3이 아닌 문자열
- 숫자가 아닌 문자열
- 중복되는 숫자가 들어오는 경우
- 게임이 끝난 후 1, 2가 아닌 이외의 값이 들어오는 경우

## 구현한 기능 목록

------
## - Class computer
> 게임의 생성, 진행, 종료를 관리하며 규칙을 관리하는 객체 
> 
> initGame : 게임을 시작한 후 랜덤한 숫자를 부여한다.
> 
> getPlayGame : 게임의 종료 여부를 알려주는 함수
> 
> determineQuery : 사용자의 질의를 판단하여 응답을 출력한다.
